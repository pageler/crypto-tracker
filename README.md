# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

# Crypto Tracker
## Purpose:
* Easy resource to follow coins, current value, percent change, and market caps.

## Components:
* Header with dropdown menu for US dollars, British pounds, and Euro valuations.
* Header login button with LOGIN, SIGN UP, and Sign In With Google options.
* Carousel featuring trending coins with coin image, coin symbol, 24 hour percent change, and current price.
* Search bar
* Coin List sorted by market cap that is clickable to show detailed information about each coin.
* Pagination component at the bottom of the page.
* Single coin detail page shows market information on the left and an interactive graph on the right.
* Coins can be added or deleted from watchlist.
